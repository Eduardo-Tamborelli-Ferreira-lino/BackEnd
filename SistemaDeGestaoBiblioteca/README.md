# 📚 Sistema de Gestão de Biblioteca (Legacy)

Versão inicial de um sistema de biblioteca desenvolvido sem o uso de frameworks de automação de build como Maven, focando na organização manual de pacotes e dependências.

## 🛠️ Tecnologias
- **Java**
- **JDBC**
- **SQL**

## 🏗️ Estrutura de Pacotes
- `Model`: Entidades (Autor, Livro).
- `DAO`: Persistência de dados.
- `DTO`: Objetos de transferência de dados.
- `Connection`: Gerenciamento de drivers e conexões.
- `Main`: Ponto de entrada do sistema.
