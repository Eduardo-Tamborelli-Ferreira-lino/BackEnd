# ✈️ AeroManager

Sistema de gestão de tráfego aéreo e aeroportos, desenvolvido para gerenciar frotas, voos e passageiros.

## 🛠️ Tecnologias
- **Java**
- **JDBC**
- **Maven**
- **Padrão Repository**

## 🏗️ Entidades Principais
- **Airport**: Gestão de aeroportos e localizações.
- **Flight**: Controle de horários, rotas e aeronaves.
- **Plane**: Cadastro de aeronaves e capacidades.
- **Passenger**: Gestão de dados dos passageiros.
- **Reservation**: Vínculo entre passageiros e voos.

## 🚀 Como Executar
1. Certifique-se de configurar o banco de dados conforme o arquivo `SQL.txt`.
2. Compile o projeto:
   ```bash
   mvn clean install
   ```
3. Execute a classe `Main`.
