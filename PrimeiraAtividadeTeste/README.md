# 🧪 Primeira Atividade de Testes

Primeiro contato prático com o desenvolvimento orientado a testes utilizando **Java** e **Docker**.

## 🛠️ Tecnologias
- **Java**
- **JUnit 5**
- **Docker**
- **Maven**

## 🚀 O que foi aprendido
- Configuração de ambiente de teste com containers.
- Escrita de casos de teste para métodos de serviço.
- Ciclo básico de Red-Green-Refactor.
