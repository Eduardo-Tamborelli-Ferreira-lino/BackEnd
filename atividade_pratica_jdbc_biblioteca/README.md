# 📚 Atividade Prática JDBC - Biblioteca

Exercício de modelagem e implementação de um sistema de biblioteca utilizando **JDBC**.

## 🛠️ Tecnologias
- **Java**
- **JDBC**
- **Maven**
- **SQL**

## 🚀 Foco da Atividade
- Modelagem de banco de dados relacional.
- Implementação de DAOs para `Livro`, `Autor` e `Emprestimo`.
- Manipulação de datas e estados de objetos.
