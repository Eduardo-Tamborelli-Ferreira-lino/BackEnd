# 🏫 Biblioteca — Aula de Testes

Projeto desenvolvido durante aula prática para exercitar a aplicação de **testes unitários** e o **padrão Repository**.

## 🎯 Objetivo
Aprofundar o conhecimento em JUnit 5 e Mockito, garantindo que a lógica de negócio esteja protegida por testes automatizados.

## 🛠️ Tecnologias
- **Java**
- **JUnit 5**
- **Mockito**
- **Maven**

## 🏗️ Estrutura
- `repository`: Interfaces e implementações para acesso a dados.
- `service`: Camada de serviço onde a lógica de negócio é testada.
- `model`: Entidades de domínio.
